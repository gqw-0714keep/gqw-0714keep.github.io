搭建个人博客

2022/5/17 博客 hexo

一个简单的搭建博客教程

1.环境配置

首先需要安装Node.js和Git（点击跳转）

cmd输入node -v检查node.js，正常显示出版本号即安装成功

检查git：一般在任意文件夹或桌面的空白处右键即可显示Git GUI Here 和Git Bash Here

然后打开cmd全局下载hexo脚手架：npm install hexo-cli -g

//附带一个淘宝镜像cnpm
npm i -g cnpm --registry=https://registry.npm.taobao.org

2.搭建框架

选中任意盘的根目录，使用右键打开Git Bsh Here

在命令函中输出hexo init blog

这将会创建一个名为blog的文件夹，可以任意命名，这里也可以移步至Hexo的官方文档

cd blog进入刚刚创建的文件夹下载依赖npm install

//hexo三连
$ hexo cl
$ hexo g
$ hexo s

hexo cl清除已渲染配置成功的文件，hexo g重新加载并渲染生成配置文件，hexo s启动本地服务，默认端口为4000，可以通过http://localhost:4000/进行查看(例如我写文章时，通常在本地查看一遍再推送到github上)

3.创建Github仓库

打开Github网站进行注册或者登录

打开我们的存储库并新建一个，如下图：

在新建时将我们的Repository name填写为

<自己Github的username>.github.io

例如我的：gqw-0714keep.github.io

之后点击创建仓库（Create repository），该仓库名会作为域名来在线访问你的博客

4.连接Github

打开Git Bash，设置要连接的用户名和邮箱（全局的）

$ git config --global user.name "GitHub 用户名"
$ git config --global user.email "GitHub 邮箱"

SSH密钥：

输入ssh-keygen -t rsa -C "GitHub 邮箱"，一直回车就好

在C:\Users\用户名中存在**.ssh文件夹，使用记事本打开id_rsa.pub**文件并复制里面的内容（没有请勾选显示隐藏的项目）

在github的仓库中打开Deploy keys添加一个新的密钥，将内容粘贴在key中

在Git Bash输入ssh -T git@github.com,出现”Are you sure”，输入yes回车，显示“Hi xxx! You ve successfully….” 即连接成功

5.部署Hexo至GitHub Pages

安装 hexo-deployer-git：npm install hexo-deployer-git --save

打开blog根目录下的**_config.yml**文件，修改末尾部分

deploy:
  type: git
  repository: git@github.com:用户名/用户名.github.git
  branch: master

hexo d进行推送，访问https://用户名.github.io（请等待片刻，github资源刷新较慢）
