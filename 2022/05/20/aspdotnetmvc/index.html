如何使用MVC的Layout布局页

2022/5/20 学习笔记 mvc

.Net Core MVC学习笔记

什么是布局页

每次新建ASP .NET MVC 项目中(.netframework)，都会存在一个Shared文件夹用来存放我们的 Layout 布局页，系统将会生成一个默认布局

举例：在浏览同一种内容时，网页的头部+侧边菜单+脚部都是不会变的，作为固定样式，只有中间的内容页会发生改变

如何自定义布局页

首先右键在Shared文件中新建一个布局页，在需要改变内容的位置添加@RenderBody，例如：

<div>
  @RenderBody
</div>

解释一下：将布局页看做母页，嵌入该位置的html页面看做子页，只要绑定了该布局页(母页)，对子页进行访问时就会插入到指定位置显示

在子页的头部设置母页的路径即可：

@{
    ViewBag.Title = "xxx";
    Layout = "~/Views/Shared/_BackAdmin.cshtml";
}

同时也要在Views文件夹下的**_ViewStart.cshtml**中做出对应的修改

不想使用布局页时，设置为null

@{
    ViewBag.Title = "xxx";
    Layout = null;
}
