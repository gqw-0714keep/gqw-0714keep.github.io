Git指令

2022/5/22 学习笔记 git

一些常用的git指令

Git 全局设置

git config --global user.name "xxx"
git config --global user.email "xxx"

添加远程仓库

git remote add origin <仓库地址>

删除远程仓库

git remote rm origin

修改远程仓库

git remote set-url --push [name][newUrl]

查看当前连接的远程仓库信息

git remote -v

初始化仓库

git init

拷贝一份远程仓库（下载项目）

git clone

添加文件到暂存区/缓存区

git add

将暂存区内容添加到仓库中

git commit

创建远程分支

//git push 仓库名 分支名:分支名
git push origin dev:dev

查看远程分支/查看本地分支/查看所有分支

git branch -r
git branch 
git branch -a

删除远程分支

//可以推送一个空分支到远程分支，也可以直接删除
git push origin :dev
git push origin --delete dev
